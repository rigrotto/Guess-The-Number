# Guess The Number
A simple command-line game where you try to guess a randomly generated number between 1 and 100.

# Overview
* The game picks a random number between 1 and 100
* You enter guesses in the terminal
* The game tells you if your guess is too high or too low
* It keeps going until you guess the correct number
* At the end, it shows how many attempts you took
# How To Play
Head to [the website](https://gtn.pages.dev/).
# Example Gameplay
```
Welcome to Guess the Number!
I'm thinking of a number between 1 and 100. Can you guess it?

Take a guess: 50
Too low! Try again.

Take a guess: 75
Too high! Try again.

Take a guess: 63
Congratulations! You guessed the number in 3 attempts.
```
# Features
* Random number generation
* Unlimited guesses
* Tracks number of attempts
* Simple and beginner-friendly
